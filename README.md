# Usage

1.  Install following tools:
    - [Terraform](https://terraform.io/)
    - [Yandex Cloud CLI](https://yandex.cloud/en/docs/cli/)
    - [VSCode](https://code.visualstudio.com/)
2.  Create and populate `.env` file using `.env.example` as a reference
3.  Run task `TF Init`
4.  Run task `TF Apply`
5.  Enjoy
